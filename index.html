Numeral Engine V10

2D Engine:
• Size format: X-Y (example: 4-4)
• Binary/Decimal/Hex/Octal/Morse require R G B values per pixel
• ColorCode requires #RRGGBB per pixel
• Stress Mode generates correct amount automatically

3D Engine:
• Size format: X-Y-Z (example: 3-3-3)
• Edit Cube → Full solid voxel grid (limit 10-10-10)
• Edit Faces → Hollow surface only (limit 25-25-25)
• Stress Mode auto-fills correct surface/volume count
• Transparency slider affects all voxels
• Swipe to rotate • Pinch to zoom
