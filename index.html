Numeral Engine

2D Engine Instructions:
• Size format: X-Y (example: 4-4 means width 4 pixels, height 4 pixels)
• Binary / Morse require 8 characters per value
• Decimal range: 0–255
• Hex range: 00–FF
• Octal max: 377
• ColorCode format: #RRGGBB
• Stress Mode generates random valid values automatically

3D Engine Instructions:
• Size format: X-Y-Z (width-height-depth)
• Example: 3-2-5
• Swipe to rotate
• Pinch to zoom
• Transparency slider changes opacity
